# Amazon Front-End Clone

**Live Demo:** [https://vikrantssingh.github.io/amazon/](https://vikrantssingh.github.io/amazon/)

A beginner-friendly front-end clone of the Amazon homepage built using HTML and CSS. This project focuses on replicating the core UI elements of Amazon, such as the navigation bar, search section, and product display grids, to practice layout building and styling.

## Features

- **Header & Navigation Bar**: Replicates the classic Amazon navigation bar with the logo, delivery location, search bar, and cart icon.
- **Secondary Navigation**: Includes the sub-navigation menu for categories like "Today's Deals", "Mobiles", "Best Sellers", etc.
- **Hero Image Section**: A large banner image typical of e-commerce homepages.
- **Product Grids**: Layout for displaying product categories and items using CSS flexbox or grid.
- **Amazon Brand Colors**: Uses standard Amazon UI hex colors (e.g., `#0f1111`, `#232f3e`, `#febd69`, and `#eaeded`) for an authentic look and feel.

## Technologies Used

- **HTML5**: For structuring the web page and its various components.
- **CSS3**: For styling, responsive layouts, and incorporating brand colors and fonts.
- **FontAwesome**: Used for incorporating standard icons (cart, search, location, etc.).

## How to Run

Since this is a static website, no complex setup or local server is required. 

1. Clone or download the repository to your local machine.
2. Open the project folder.
3. Double-click on `index.html` to open it in your default web browser.
   *(Alternatively, right-click `index.html` and select "Open with" to choose a specific browser like Chrome, Firefox, or Edge).*

## Customization & Improvements Made

- **Updated Color Palette**: Adjusted CSS to use accurate Amazon brand colors instead of default placeholder colors (like pink or deep purple), making the UI look much more professional and realistic.
- **Refined Styling**: Minor styling tweaks applied to the search bar and navigation elements for a cleaner appearance.
